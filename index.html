About

The aim of bcstack.org is to create royalty-free software that can be used to make Bluetooth SMART products. Its core is an ultra lightweight Bluetooth Low Energy (BLE) single mode stack.

License

bcstack is available under Apache V2 license. It can be used freely in commercial products.

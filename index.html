bcstack.org

An embedded, royalty-free, lightweight Bluetooth SMART software stack

About

The aim of bcstack.org is to create royalty-free software that can be used to make Bluetooth SMART products. Its core is an ultra lightweight Bluetooth Low Energy (BLE) single mode stack.

License

bcstack is available under Apache V2 license. It can be used freely in commercial products.

Supported Platforms

bcstack works on linux with a 4.0 USB Bluetooth Dongle but it is designed to work on microcontrollers.

Why we are doing this?

We see most (if not all) bluetooth le stacks are not open sourced so we cannot touch its internals and hack it for fun, so we started to develop an open source stack.

How to help

If you are also interested in developing an open source bluetooth smart stack, join us and help. You may:

  • spread the idea and help to find more people to join
  • hack the code and help us to improve it
  • tell us your interesting ideas
